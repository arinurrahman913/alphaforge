# AlphaForge Core Values

---

# 1. Truth Over Opinion

Investment decisions must be supported by evidence.

Opinions are welcome.

Evidence is required.

---

# 2. Knowledge Before Software

Knowledge is the primary asset.

Software exists to organize knowledge.

Technology may change.

Knowledge compounds.

---

# 3. Evidence Before Emotion

Investment decisions should be based on facts, not excitement, fear, or market noise.

---

# 4. Long-Term Thinking

AlphaForge is optimized for long-term decision quality rather than short-term market prediction.

---

# 5. Continuous Improvement

Every research project...

Every playbook...

Every document...

Every AI agent...

should improve over time.

---

# 6. Documentation Is Development

If knowledge is not documented, it cannot compound.

Documentation is considered part of development.
