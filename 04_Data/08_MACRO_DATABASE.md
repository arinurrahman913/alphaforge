# AlphaForge Macro Database

## Purpose

The Macro Database stores structured information about global and regional economic conditions.

Its purpose is to provide macroeconomic context that influences markets, sectors, industries, and companies.

Macroeconomic trends shape capital allocation, valuation, and long-term investment opportunities.

---

# Core Identity

Every macro record contains:

- Country
- Region
- Reporting Period
- Data Source

# AlphaForge Principle

Macro does not predict every company.

Macro defines the environment in which companies operate.

Understanding macro trends improves investment decisions by providing context before analysis.
