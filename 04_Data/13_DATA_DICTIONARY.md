# AlphaForge Data Dictionary

## Purpose

Every data field should have a clear definition.

This prevents ambiguity across the platform.

---

# Standard Definition

Every field contains:

- Name
- Description
- Data Type
- Unit
- Source
- Example

---

# Naming Convention

Examples:

Revenue

Operating Margin

Free Cash Flow

ROE

ROIC

Market Cap

Enterprise Value

---

# AlphaForge Principle

Shared definitions create consistent intelligence.
